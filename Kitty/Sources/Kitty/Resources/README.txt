Drop the three cat PNGs in this folder, with these exact filenames:

  cat_idle.png       — shown when the cat is just hanging out      (the bow cat)
  cat_listening.png  — shown while the mic is open (you're talking) (the tongue-out cat)
  cat_busy.png       — shown while thinking and answering           (the tie cat)

Square images with a transparent background work best (~256×256 or larger).
PNG is required so the black background can be transparent. If you have JPGs
you can convert in Preview: File → Export → Format: PNG → tick "Alpha".

If a file is missing the app falls back to a colored circle so it still runs —
you'll just see a flat dot instead of a cat.
